ImageCrop.py must be used to prepare all training and test images, images are then saved in dataset folder within image crop folder.
To run recognition software open command prompt enter *File Path*\AICourseWork\x64\Debug\AICourseWork.exe 
followed by File paths for both csv files containging location of test and training dataset.

EXAMPLE - C:\Users\ryanf\Documents\AICourseWork\Code\x64\Debug\AICourseWork.exe C:\Users\ryanf\Documents\AICourseWork\Code\csvFile.csv C:\Users\ryanf\Documents\AICourseWork\Code\csvFileTest.csv
